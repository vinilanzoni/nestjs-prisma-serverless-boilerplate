# NestJS Prisma Serverless Boilerplate

This project uses the NestJS framework to create a serverless application using AWS Lambda and API Gateway.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# serverless offline mode
$ npm run start:offline
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
